# 첫 포스팅 기술블로그 작성

1. 기술블로그 작성 목적

   - 커리어 관리
   - 소통능력 향상
   - 성실함 어필

2. 기술블로그 작성 방법

   가. 새로운 기술을 습득한 경우

   - 필요한 이유 
   - 매뉴얼화(누가봐도 쉽게, 누가봐도 친절하게, 윈도우와 맥을 함께)

   나. 새로운 해결 방법을 습득한 경우

   - 이슈내용
   - 해결방법(그때그때 바로 작성할 것)

   다. 프로젝트 회고

   - KPT 방법론
     - 긍정적 요소는 KEEP
     - PROBLEM 은 개선
     - P를 줄이기 위한 TRY
   - 5F 방법론
     - FACT/FEELING/FEEDBACK/FINDING/FUTURE ACTION을 
     - 프로젝트 단계를 시간대로 나누어 5F가 포함되도록 평가
